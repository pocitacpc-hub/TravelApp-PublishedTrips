# Patche

Patch model je plánovaná navazující vrstva. Základní publikovaný program zůstane neměnným vstupem a změny budou cílit na stabilní `publicId`.

Verze 1.0 má lokálně podporovat odškrtnutí, přeskočení, změnu času, přesun, poznámku a přidání aktivity. Lokální změny se uloží do IndexedDB telefonu a půjdou exportovat/importovat. Publikování patchů přes GitHub Actions patří do verze 1.1.
